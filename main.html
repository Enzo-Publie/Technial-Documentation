Introduction

The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.

Html can be assisted by other technologies such as for example:

  • Cascading Style Sheets (CSS)
  • Scripting languages such as JavaScript

Where to start

There are many ways to learn HTML if you are new to the language or programming in general.

Some of the best places to learn are the following:

  • FreeCodeCamp
  • W3schools
  • Youtube

Hello World

To get started with writing HTML, open notepad or any other text editor and write your first "Hello world" html code:

<p>Hello World</p>

If you save the document as a html file and open it, you will see the text Hello World displayed in your browser of choice.

Elements

HTML documents imply a structure of nested HTML elements. These are indicated in the document by HTML tags, enclosed in angle brackets like <p>

In the simple, general case, the extent of an element is indicated by a pair of tags: a "start tag" <p> and "end tag" </p>. The text content of the element, if any, is placed between these tags.
